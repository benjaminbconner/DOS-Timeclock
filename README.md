# DOS Timeclock ⏱️

A simple, retro-styled **web-based timeclock** that mimics the look and feel of old DOS terminals.  
Employees can clock in/out, admins can manage users, and all data is stored locally in the browser via `localStorage`.

---

## ✨ Features
- **Clock In/Out** with name + PIN
- **Employee status dashboard** (who’s in/out, hours worked today)
- **Event log** of all punches
- **Admin tools**:
  - Add employees
  - Export daily events to CSV
  - Reset today’s log
  - Clear all data
- **Retro DOS-style UI** with green phosphor text
- **Single-file app** — just open `timeclock.html` in your browser
